典型值范围：
- 浊音：50-250 Hz
- 清音：300-3000 Hz

频域特征分析

基频(F0) 基频代表声音的基本音高，是声带振动的频率。

典型值范围：
- 男声：85-180 Hz
- 女声：165-255 Hz
- 儿童：250-400 Hz
基频越高，感知的音高越高。 0 Hz
音高基准频率
谐波数 谐波是基频的整数倍频率，反映了声音的音色丰富程度。

典型值范围：4-12个
谐波数越多：
- 音色越丰富
- 声音听起来越饱满
- 元音发音越清晰 0
音色丰富度
频谱质心 频谱质心表示声音能量分布的"重心"位置，反映音色的明亮度。

典型值范围：
- 低沉声音：500-1500 Hz
- 明亮声音：1500-3000 Hz
- 尖锐声音：3000+ Hz
质心频率越高，声音听起来越明亮。 0 Hz
